#+TITLE: Equivalencia entre Haskell y Python

* Funciones aritméticas

** Funciones sobre números

|------------+-----------|
| Haskell    | Python    |
|------------+-----------|
| x + y      | x + y     |
| x - y      | x - y     |
| x * y      | x * y     |
| x / y      | x / y     |
| x ^ y      | x ** y    |
| max x y    | max(x, y) |
| minimum xs | min(x)    |
| maximum xs | max(xs)   |
|------------+-----------|

** Funciones sobre enteros

|---------+--------|
| Haskell | Python |
|---------+--------|
| rem a b | a % b  |
|---------+--------|

** Funciones sobre reales

|---------+---------|
| Haskell | Python  |
|---------+---------|
| pi      | math.pi |
|---------+---------|

* Funciones sobre listas

** Funciones elementales

|-----------+---------|
| Haskell   | Python  |
|-----------+---------|
| xs ++ ys  | xs + ys |
| head xs   | xs[0]   |
| tail xs   | xs[1:]  |
| take n xs | xs[n:]  |
| drop n xs | xs[:n]  |
|-----------+---------|
